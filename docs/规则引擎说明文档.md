## 一、数据预处理说明

在执行规则前，需对原始数据进行以下清洗与结构化处理：

### 1. 金额字段转换
将"借方本币"和"贷方本币"中的逗号去除，转换为浮点数。

```python
for col in ['借方本币', '贷方本币']:
    df[col] = df[col].astype(str).str.replace(',', '').astype(float)
```

### 2. 辅助项解析
将辅助项文本解析为结构化字典，提取关键字段：

| 字段 | 格式 | 提取示例 |
|------|------|---------|
| 银行账户 | `【银行账户：...】` | 东方农商行营业部0868 |
| 金融机构 | `【金融机构：...】` | 中国建设银行 |
| 部门 | `【部门：...】` | 综合办公室 |
| 合同号 | `【合同号：...】` | 03-GY001-9999 |
| 增值税税码税率 | `【增值税税码税率：...】` | 6% |

```python
def parse_auxiliary(aux_text):
    if pd.isna(aux_text):
        return {}
    matches = re.findall(r'【(.*?)：(.*?)】', str(aux_text))
    return {k.strip(): v.strip() for k, v in matches}

df['辅助项_dict'] = df['辅助项'].apply(parse_auxiliary)
```

### 3. 凭证号数字部分
提取"记-0001"中的数字"1"用于断号检查。

```python
df['凭证号数字'] = df['凭证号'].str.extract(r'(\d+)').astype(float)
```

### 4. 科目层级拆分
从科目名称中提取各级科目：

```python
df['科目末级'] = df['科目名称'].apply(lambda x: str(x).split('\\')[-1])
df['科目一级'] = df['科目名称'].apply(lambda x: str(x).split('\\')[0])
```

## 二、凭证级规则

### 规则1：借贷不平

| 属性 | 值 |
|------|-----|
| **风险等级** | 高 |
| **业务背景** | 凭证借贷方金额不相等，属于严重会计错误 |

**检测逻辑**：
1. 按凭证号分组
2. 计算借方合计与贷方合计
3. 若差额绝对值大于容差，则标记该凭证所有分录

**配置参数**：
```yaml
params:
  balance_tolerance: 0.01  # 差额容差
```

**代码实现**：
```python
def _check_balance(self):
    config = self.rule_config.get('凭证级规则', {}).get('借贷不平', {})
    tolerance = config.get('params', {}).get('balance_tolerance', 0.01)
    
    balance = self.df.groupby('凭证号').agg({
        '借方本币': 'sum', 
        '贷方本币': 'sum'
    }).reset_index()
    
    balance['差额'] = (balance['借方本币'] - balance['贷方本币']).abs()
    unbalanced = balance[balance['差额'] > tolerance]['凭证号'].tolist()
    
    mask = self.df['凭证号'].isin(unbalanced)
    for idx in self.df[mask].index:
        voucher = self.df.loc[idx, '凭证号']
        diff = balance[balance['凭证号'] == voucher]['差额'].values[0]
        self._add_risk([idx], '借贷不平', '高', f'借贷差额{diff:.2f}超过容差{tolerance}')
    
    logger.info(f"借贷不平: {mask.sum()} 条")
```

**输出示例**：
```
凭证号"记-0001"借方合计30,000,000.00，贷方合计30,000,001.00，差额1.00
```

```python
def _check_voucher_sequence(self):
    config = self.rule_config.get('凭证级规则', {}).get('凭证断号', {})
    exclude = config.get('params', {}).get('exclude_numbers', [])
    
    voucher_nums = self.df['凭证号数字'].dropna().unique()
    voucher_nums = sorted(set(voucher_nums) - set(exclude))
    
    if len(voucher_nums) > 1:
        gaps = []
        for i in range(1, len(voucher_nums)):
            if voucher_nums[i] - voucher_nums[i-1] > 1:
                gap_start = voucher_nums[i-1] + 1
                gap_end = voucher_nums[i] - 1
                gaps.append(f"{gap_start}-{gap_end}" if gap_start != gap_end else str(gap_start))
        
        if gaps:
            self._add_risk(self.df.index[:1], '凭证断号', '中', f'凭证号缺失: {", ".join(gaps)}')
            logger.info(f"凭证断号: {len(gaps)} 处")
```

**输出示例**：
```

```

### 规则3：一借多贷/一贷多借异常

| 属性 | 值 |
|------|-----|
| **风险等级** | 中 |
| **业务背景** | 同一凭证中，一笔借方对应多笔贷方（或反之），且贷方金额完全相同、科目相同、辅助项中的银行账户相同，可能为人为拆分支付以规避审批 |

**检测逻辑**：
1. 筛选借方笔数=1且贷方笔数>1（或反向）的凭证
2. 检查贷方分录的金额是否全部相等（容差内）
3. 若贷方科目均为"银行存款"，且辅助项中的"银行账户"完全相同，则标记为高风险拆分
4. 若科目不同但属于同一大类（如费用类），且摘要相似，标记为中风险

**配置参数**：
```yaml
params:
  amount_tolerance: 0.01           # 金额容差
  same_bank_account_required: true # 银行账户相同才标记
```

**代码实现**：
```python
def _check_one_debit_multi_credit(self):
    config = self.rule_config.get('凭证级规则', {}).get('一借多贷异常', {})
    params = config.get('params', {})
    amount_tol = params.get('amount_tolerance', 0.01)
    same_bank = params.get('same_bank_account_required', True)
    
    for voucher in self.df['凭证号'].unique():
        v_df = self.df[self.df['凭证号'] == voucher]
        debit_count = (v_df['借方本币'] > 0).sum()
        credit_count = (v_df['贷方本币'] > 0).sum()
        
        # 检查一借多贷
        if debit_count == 1 and credit_count > 1:
            credit_df = v_df[v_df['贷方本币'] > 0]
            amounts = credit_df['贷方本币'].values
            
            # 检查金额是否相等
            if amounts.max() - amounts.min() <= amount_tol:
                # 检查银行账户
                bank_accounts = [v_df.loc[idx, '辅助项_dict'].get('银行账户', '') 
                                for idx in credit_df.index]
                
                if same_bank and len(set(bank_accounts)) == 1:
                    self._add_risk(v_df.index, '一借多贷异常', '中', 
                                  f'1借{credit_count}贷，金额均为{amounts[0]:.2f}，银行账户相同')
        
        # 检查一贷多借（反向逻辑）
        if credit_count == 1 and debit_count > 1:
            # ... 类似逻辑
            pass
```

**输出示例**：
```
凭证号"记-0001"为1借3贷，贷方3笔均为10,000,000.00元，且银行账户均为"东方农商行营业部0868"，可疑拆分
```

### 规则8：进项税与费用匹配性检查

| 属性 | 值 |
|------|-----|
| **风险等级** | 中 |
| **业务背景** | 进项税应与其对应的费用分录在同一凭证中，且金额逻辑合理（如费用金额×税率≈税额） |

**检测逻辑**：
1. 在同一凭证内，筛选进项税分录和费用类分录
2. 按摘要相似度分组，将进项税与费用关联
3. 计算费用金额×适用税率，与进项税金额比较
4. 若差异超过阈值则标记

**配置参数**：
```yaml
params:
  tax_rate_tolerance: 0.01   # 税额差异容差
  summary_similarity: 0.8    # 摘要相似度阈值
```

**代码实现**：
```python
def _check_tax_expense_match(self):
    config = self.rule_config.get('科目合规规则', {}).get('进项税费用匹配', {})
    params = config.get('params', {})
    tolerance = params.get('tax_rate_tolerance', 0.01)
    
    for voucher in self.df['凭证号'].unique():
        v_df = self.df[self.df['凭证号'] == voucher]
        
        # 筛选进项税分录
        tax_mask = v_df['科目编码'].str.startswith('2221', na=False)
        tax_df = v_df[tax_mask]
        
        # 筛选费用分录
        expense_mask = v_df['科目名称'].str.contains('费|支出', na=False)
        expense_df = v_df[expense_mask]
        
        if len(tax_df) > 0 and len(expense_df) > 0:
            for _, tax_row in tax_df.iterrows():
                tax_amount = tax_row['贷方本币'] if tax_row['贷方本币'] > 0 else tax_row['借方本币']
                tax_rate_str = tax_row['辅助项_dict'].get('增值税税码税率', '0%')
                tax_rate = float(re.search(r'(\d+)', tax_rate_str).group(1)) / 100 if re.search(r'(\d+)', tax_rate_str) else 0
                
                # 查找对应费用
                for _, exp_row in expense_df.iterrows():
                    exp_amount = exp_row['借方本币'] if exp_row['借方本币'] > 0 else exp_row['贷方本币']
                    expected_tax = exp_amount * tax_rate
                    
                    if abs(expected_tax - tax_amount) > tolerance * exp_amount:
                        self._add_risk([tax_row.name], '进项税费用匹配', '中',
                                      f'税额{tax_amount:.2f}与费用{exp_amount:.2f}×税率{tax_rate:.0%}={expected_tax:.2f}不符')
```

**输出示例**：
```
凭证号"记-0002"中，费用分录3金额1,419.00，税率6%应计提税额85.14，但进项税分录5、6、7合计80.31，差异较大
```

### 规则11：频繁小额交易

| 属性 | 值 |
|------|-----|
| **风险等级** | 低 |
| **业务背景** | 同一部门或合同号下频繁发生小额费用，可能为拆分报销 |

**检测逻辑**：
1. 按"部门"或"合同号"分组
2. 在设定时间窗口内统计小额交易次数
3. 超过阈值则标记

**配置参数**：
```yaml
params:
  small_amount_threshold: 1000        # 小额阈值
  time_window_days: 30                # 时间窗口（天）
  count_threshold: 5                  # 次数阈值
  group_by: ['部门', '合同号']         # 分组字段
```

**代码实现**：
```python
def _check_frequent_small(self):
    config = self.rule_config.get('金额合理性规则', {}).get('频繁小额', {})
    params = config.get('params', {})
    small_threshold = params.get('small_amount_threshold', 1000)
    count_threshold = params.get('count_threshold', 5)
    group_by = params.get('group_by', ['部门'])
    
    small_df = self.df[self.df['amount'] < small_threshold]
    
    for field in group_by:
        if field in self.df.columns or all(field in d for d in self.df['辅助项_dict']):
            # 提取分组字段值
            if field in self.df.columns:
                small_df['group_key'] = small_df[field]
            else:
                small_df['group_key'] = small_df['辅助项_dict'].apply(lambda x: x.get(field, ''))
            
            grouped = small_df[small_df['group_key'] != ''].groupby('group_key')
            
            for name, group in grouped:
                if len(group) >= count_threshold:
                    self._add_risk(group.index, '频繁小额', '低',
                                  f'{field}="{name}"有{len(group)}笔小额交易')
```

**输出示例**：
```
综合办公室在30天内发生差旅费小额报销8笔，建议检查是否拆分
```

```python
def _check_split_by_bank_account(self):
    config = self.rule_config.get('金额合理性规则', {}).get('异常拆分', {})
    min_count = config.get('params', {}).get('same_amount_min_count', 3)
    
    for voucher in self.df['凭证号'].unique():
        v_df = self.df[self.df['凭证号'] == voucher]
        bank_df = v_df[v_df['科目名称'].str.contains('银行存款', na=False)]
        
        # 按银行账户分组
        for bank_account, group in bank_df.groupby(
            lambda idx: self.df.loc[idx, '辅助项_dict'].get('银行账户', '')
        ):
            if bank_account:
                amounts = group['amount'].values
                unique, counts = np.unique(amounts, return_counts=True)
                
                for amt, cnt in zip(unique, counts):
                    if cnt >= min_count:
                        self._add_risk(group.index, '异常拆分', '高',
                                      f'银行账户"{bank_account}"有{cnt}笔相同金额{amt:,.2f}')
```

**输出示例**：
```
凭证号"记-0001"中对东方农商行账户有3笔10,000,000.00元支付，异常拆分
```

1. 按合同号分组

```python
def _check_contract_period(self):
    config = self.rule_config.get('辅助项专项规则', {}).get('合同跨期异常', {})
    inactive_days = config.get('params', {}).get('contract_inactive_days', 365)
    
    # 提取合同号
    self.df['合同号'] = self.df['辅助项_dict'].apply(lambda x: x.get('合同号', ''))
    contract_df = self.df[self.df['合同号'] != '']
    
    grouped = contract_df.groupby('合同号').agg({
        'date': ['min', 'max'],
        'amount': 'sum'
    })
    
    today = datetime.now()
    for contract, row in grouped.iterrows():
        last_date = row['date']['max']
        if (today - last_date).days > inactive_days:
            mask = self.df['合同号'] == contract
            self._add_risk(self.df[mask].index, '合同跨期异常', '中',
                          f'合同"{contract}"最后费用日期{last_date.strftime("%Y-%m-%d")}，已{inactive_days}天无新费用')
```

### 规则18：月末/年末突击记账

| 属性 | 值 |
|------|-----|
| **风险等级** | 中 |
| **业务背景** | 期末大量记账可能调节利润 |

**检测逻辑**：
1. 统计每月最后N天的记账笔数和金额占比
2. 若超过当月总额的阈值则标记

**配置参数**：
```yaml
params:
  last_days: 5           # 期末天数
  ratio_threshold: 0.4   # 占比阈值
```

**代码实现**：
```python
def _check_month_end_rush(self):
    config = self.rule_config.get('时间异常规则', {}).get('月末突击', {})
    params = config.get('params', {})
    last_days = params.get('last_days', 5)
    ratio_threshold = params.get('ratio_threshold', 0.4)
    
    self.df['month'] = self.df['date'].dt.to_period('M')
    self.df['day'] = self.df['date'].dt.day
    self.df['days_in_month'] = self.df['date'].dt.days_in_month
    
    for month, group in self.df.groupby('month'):
        total_amount = group['amount'].sum()
        month_end_mask = group['day'] > (group['days_in_month'].iloc[0] - last_days)
        month_end_amount = group[month_end_mask]['amount'].sum()
        
        ratio = month_end_amount / total_amount if total_amount > 0 else 0
        
        if ratio > ratio_threshold:
            self._add_risk(group[month_end_mask].index, '月末突击', '中',
                          f'{month}最后{last_days}天记账金额占全月{ratio:.0%}')
```

**输出示例**：
```
2021年1月最后5天记账金额占全月60%，异常
```
